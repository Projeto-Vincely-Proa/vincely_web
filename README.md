# Vincely 
Plataforma que conecta comunidades para oferecer apoio emocional a pessoas em situações de vulnerabilidade, lidando com ansiedade, depressão e crises de pânico, promovendo saúde mental e bem-estar.

## Scripts Disponíveis

No diretório do projeto, você pode executar:

`npm start`

Executa a aplicação em modo de desenvolvimento.
Abra http://localhost:3000
 para visualizar no navegador.

A página será recarregada automaticamente sempre que houver alterações no código.
Você também poderá ver erros de lint no console.

---
`npm test`

Inicia o executor de testes em modo interativo (watch mode).
Para mais informações, consulte a seção sobre
execução de testes
.

---
`npm run build`

Gera a versão de produção da aplicação na pasta build.

O React é corretamente empacotado em modo de produção e o build é otimizado para melhor desempenho.

Os arquivos gerados são minificados e seus nomes incluem hashes.
Sua aplicação estará pronta para deploy.

Para mais informações, consulte a seção sobre
deploy da aplicação
.

---
`npm run eject`

⚠️ Atenção: esta é uma operação irreversível. Após executar eject, não é possível voltar atrás.

Caso você não esteja satisfeito com as configurações e ferramentas padrão, é possível executar o comando eject a qualquer momento.
Esse comando remove a dependência única de build do projeto.

Em vez disso, ele copia todos os arquivos de configuração e dependências internas (Webpack, Babel, ESLint, etc.) diretamente para o projeto, permitindo total controle sobre elas.

Após o eject, todos os comandos continuarão funcionando, exceto que passarão a utilizar os scripts copiados, permitindo customizações mais profundas. A partir desse ponto, a manutenção é totalmente sua.

Você não é obrigado a utilizar o eject.
O conjunto de funcionalidades padrão é suficiente para projetos pequenos e médios. Porém, a ferramenta existe justamente para permitir customização quando necessário.
